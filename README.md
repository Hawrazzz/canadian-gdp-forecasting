# Canadian GDP Forecasting
**Harvard CS109B — Advanced Data Science | Spring 2025**

Predicting Canadian GDP using deep learning. We implement and compare three sequential models — RNN, LSTM, and a Time Series Transformer — evaluating their ability to capture GDP trends and volatility spikes.

## Models
- Naive baseline
- Recurrent Neural Network (RNN)
- Long Short-Term Memory (LSTM)
- Time Series Transformer

## Stack
Python, TensorFlow/Keras, Pandas, NumPy, Matplotlib, Scikit-learn
